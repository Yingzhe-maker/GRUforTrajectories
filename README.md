# GRUforTrajectories
A GRU model for trajectories prediction

This is a project that utilizes a variant of the RNN model and the LSTM model—the GRU (Gated Recurrent Unit) model—for trajectory prediction. The GRU model serves as the most straightforward model for time series data prediction. This project can serve as introductory learning material for beginners delving into deep learning, and can also be utilized for validation work in related research papers.
